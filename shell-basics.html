Learn how to use a computer like real programmers.

Some people get intimidated by the command line. The goal of this is to make you able a competent user and developer at the command line.

List of Common Shells

Posix Shells

  • Borne Shell (sh) - 1977
  • Bash - 1989
  • Zsh - 1990

Windows Shells

  • COMMAND.COM - 1981
  • cmd.exe - 1993
  • PowerShell - 2006

Navigating using the shell


linus@host:~$ cd Documents/Folder

The cd command changes the current directory.

Unix/Linux

Windows

Understanding the differences between:

Shell Builtins

A shell builtin is a program that is run directly within the shell.

External Programs

An external program is a program that is called by the shell.

Time to Learn Some Basic Commands!

Be Careful

The following commands can overwrite your files if used improperly.

Printing Files (to the screen)

Unix/Linux

cat file

Windows

type file

Copying Files

Unix/Linux

cp source dest

Windows

copy source dest

Renaming Files

Unix/Linux

mv old new

Windows

rename old new

Deleting Files

Unix/Linux

rm file

Windows

del file

Editing Textfiles

Unix/Linux

nano file.txt

Windows

edit file.txt

Symbolic Links (Shortcuts)

Unix/Linux

ln -s target linkname

Windows

mklink /?

Pipes are a way you can use the output of one command as the input for another.
